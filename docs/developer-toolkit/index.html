Blockchain
Fundamentals

In order to understand the blockchain, it helps to start with Bitcoin. In our first workshop, we will walk you through setting up your own Bitcoin node and using it to send transactions and record record states.

Wallet
Technology

Storing cryptographic keys properly is one of the fundamental skills required to use web3 and blockchain systems. In this module we'll teach you how to use a range of useful tools.

  • Hierarchical Deterministic Wallets
  • Seed Phrases
  • Unique Key Derivation
  • Signatures and Encryption

Second Layer
Instant Payments

Second layer technologies like Bitcoin Lightning channels provide instant payments over cryptocurrency networks.

In this workshop, we'll help you build your own store with instant Bitcoin payments!

Private
Blockchains

Some data isn't meant to be shared publicly. The Hyperledger Foundation provides a unique solution for private networks to utilize blockchain systems.

  • Network Architecture
  • Smart Contract Chaincode Design
  • Webstore Integration
  • Consensus Mechanisms

You can also read about Hyperledger in our free courses portal.

Web3, Identity,
and uPort Login

Digital Identity is a quickly expanding field allowing users to own their digital presence with personal keypairs. In this module, we teach you how to allow user login via decentralized identity, and how IPFS and the Ethereum Blockchain can be applied to make this possible.
